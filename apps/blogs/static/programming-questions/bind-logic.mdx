## bind 实现

bind() 方法创建一个新的函数，在 bind() 被调用时，这个新函数的 this 被指定为 bind() 的第一个参数，而其余参数将作为新函数的参数，供调用时使用。

```typescript
/**
 * bind() 方法创建一个新的函数，在 bind() 被调用时，这个新函数的 this 被指定为 bind() 的第一个参数，而其余参数将作为新函数的参数，供调用时使用。
 * @param target
 * @param args
 */
export default function bindLogic(source: Function, target: any, ...args: any) {
  return function (...nextArgs: any) {
    return source.call(target, ...args, ...nextArgs);
  };
}
```
