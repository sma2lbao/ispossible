### ABS 函数

返回绝对值。

#### 语法

```sql
ABS(x)
```

#### 参数说明

| 参数 | 描述                   |
| :--- | :--------------------- |
| x    | 变量。仅支持数值类型。 |

#### 示例

绝对值

```sql
SELECT ABS(-17.4);
```

返回结果：

```sql
+--------+
| abs    |
+--------+
| 17.4   |
+--------+
```

### CBRT 函数

返回立方根。

#### 语法

```sql
CBRT(dp)
```

#### 参数说明

| 参数 | 描述                   |
| :--- | :--------------------- |
| dp   | 变量。仅支持数值类型。 |

#### 示例

立方根

```sql
SELECT CBRT(27.0);
```

返回结果：

```sql
+--------+
| cbrt    |
+--------+
| 3       |
+--------+
```
